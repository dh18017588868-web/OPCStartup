# 内容策划

> 核心问题：做什么内容？怎么变现？

---

## 一、内容定位

### 1.1 一人公司内容定位原则

**先定位，再内容**

| 问题 | 答案 |
|------|------|
| 你是谁？ | 身份标签 |
| 你做什么的？ | 核心业务 |
| 凭什么信你？ | 专业背书 |
| 面向谁？ | 目标用户 |

### 1.2 内容定位公式

```
[身份] + [做什么] + [面向谁] = 内容定位
```

**示例**：

- "一个 AI 创业者，教、宝妈用 AI 做副业"
- "一个产品经理，分享 AI 工具使用技巧"
- "一个程序员，教开发者用 AI 提高效率"

### 1.3 内容风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **专业硬核** | 干货多、有深度 | 开发者、职场人 |
| **通俗易懂** | 简单、有趣 | 大众用户 |
| **案例教学** | 用案例说话 | 知识付费 |
| **真实分享** | 个人经历 | 建立信任 |

| 平台 | 用户 | 内容形式 | 变现方式 |

| **小红书** | 年轻女性 | 图文 > 视频 | 课程、带货 |

## 三、内容类型

### 3.1 按作用分类

| 类型 | 作用 |示例| 占比 |
|------|------|-----|------|
| **引流内容** | 吸引流量 | "AI 工具大全" | 40% |
| **信任内容** | 建立信任 | "我是怎么做 XXX 的" | 40% |
| **转化内容** | 引导成交 | "我的课程适合谁" | 20% |

### 3.2 按形式分类

| 形式 | 优点 | 缺点 | 适用平台 |
|------|------|------|----------|
| **图文** | 门槛低、易传播 | 曝光有限 | 小红书、知乎 |
| **短视频** | 流量大、易爆 | 制作成本高 | 抖音、视频号 |
| **长视频** | 深度、信任 | 制作门槛高 | B站 |
| **直播** | 实时互动、转化高 | 考验表现力 | 抖音、视频号 |

## 四、各平台内容策划

### 4.1 小红书

**内容特点**：

- 标题党 + 干货
- 颜值重要：封面要做好
- 情绪价值：让用户有获得感

**爆款公式**：

```
[数字] + [痛点/爽点] + [结果]
```

**示例**：

- "5 个 AI 工具，让工作效率提升 10 倍"
- "做了 3 个月自媒体，总收入 XXX"
- "普通人如何用 AI 做副业"

**选题方向**：

| 类型 | 示例 |
|------|------|
| 工具推荐 | "AI 写作工具测评" |
| 教程 | "3 分钟学会用 AI" |
| 案例 | "我是怎么做 XXX 的" |
| 避坑 | "AI 写作的 5 个误区" |
| 思维 | "AI 时代普通人机会在哪" |

### 4.2 抖音/视频号

**内容特点**：

- 前 3 秒决定生死
- 情绪化内容更容易火
- 完播率最重要

**爆款公式**：

```
[悬念/冲突/反差] + [内容] + [互动引导]
```

**选题方向**：

| 类型 | 示例 |
|------|------|
| 知识干货 | "一个技巧让 AI 写得更准" |
| 案例分享 | "我用 AI 赚了第一个 1 万" |
| 对话感 | "千万别用 AI 做这些" |
| 情绪价值 | "AI 时代普通人怎么翻身" |

### 4.3 知乎

**内容特点**：

- 深度长文
- 专业性要求高
- 搜索流量大

**爆款公式**：

```
[具体问题] + [深度解答] + [案例支撑]
```

**选题方向**：

| 类型 | 示例 |
|------|------|
| 问题回答 | "如何用 AI 提高工作效率？" |
| 深度分析 | "AI 工具的现状与未来" |
| 教程 | "从零开始学习 AI 写作" |

### 4.4 公众号

**内容特点**：

- 私域沉淀
- 长图文
- 适合深度内容

**选题方向**：

| 类型 | 示例 |
|------|------|
| 行业观察 | "2026 年 AI 创业机会" |
| 方法论 | "一人公司的 AI 变现之路" |
| 案例分析 | "一个 AI 博主的变现分析" |
| 复盘总结 | "我的第一个 10 万收入" |

## 五、内容生产

### 5.1 AI 辅助生产

**一人公司必须用 AI**：

| 环节 | AI 工具 | 作用 |
|------|---------|------|
| 选题 | Claude/GPT | 生成选题方向 |
| 标题 | Claude/GPT | 生成多个标题 |
| 写作 | Claude/GPT | 辅助写作、改写 |
| 配图 | Midjourney | 生成配图 |
| 视频 | Runway/Pika | AI 视频 |
| 翻译 | DeepL | 多语言内容 |

### 5.2 生产流程

```
1. 选题   → 确定今天发什么
    ↓
2. 标题   → 生成 5-10 个标题
    ↓
3. 大纲   → 列出要点
    ↓
4. 初稿   → AI 辅助写作
    ↓
5. 改写   → 加入个人风格
    ↓
6. 配图   → 生成或选择图片
    ↓
7. 发布   → 同步到各平台
```

### 5.3 生产效率

**一人公司内容产能建议**：

| 平台 | 最低频次 | 建议频次 |
|------|----------|----------|
| 小红书 | 3 条/周 | 1 条/天 |
| 抖音 | 3 条/周 | 1 条/天 |
| 知乎 | 1 篇/周 | 2-3 篇/周 |
| 公众号 | 1 篇/周 | 2 篇/周 |

## 六、变现路径

### 6.1 内容到变现

```
内容 → 流量 → 私域 → 成交 → 复购/裂变
```

### 6.2 变现产品

| 产品 | 特点 | 适合阶段 |
|------|------|----------|
| **引流品** | 低价/免费，扩大流量 | 早期 |
| **利润品** | 主打收入 | 成长期 |
| **高端品** | 高客单，深度服务 | 成熟期 |

### 6.3 典型变现路径

**小红书路径**：

```
笔记爆了 → 关注 → 主页 → 私域 → 课程/社群
```

**抖音路径**：

```
短视频爆 → 直播 → 转化 → 私域维护
```

**知乎路径**：

```
回答爆了 → 关注 → 付费咨询/课程
```
