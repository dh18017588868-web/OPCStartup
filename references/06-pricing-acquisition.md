# 定价获客

> 核心问题：卖多少钱？从哪里找到用户？

---

## 一、定价策略

### 1.1 一人公司定价原则

**高价策略 > 低价策略**

| 维度 | 高价策略 | 低价策略 |
|------|----------|----------|
| 用户质量 | 高 | 低 |
| 获客成本 | 低（高转化） | 高（低转化） |
| 交付压力 | 小（高毛利） | 大（薄利多销） |
| 口碑 | 好 | 差 |
| 一人公司适配 | ⭐⭐⭐⭐⭐ | ⭐⭐ |

**例外**：低价走量适用于边际成本极低的数字产品

### 1.2 定价模型

| 模型 | 说明 | 适用场景 | 一人公司 |
|------|------|----------|----------|
| **成本定价** | 成本 + 利润 | 服务类 | ⭐⭐⭐ |
| **价值定价** | 用户获益 × 愿意付费比例 | 产品类 | ⭐⭐⭐⭐⭐ |
| **竞争定价** | 比竞品低/高 | 红海市场 | ⭐⭐⭐ |
| **心理定价** | 99/199/499 | 通用 | ⭐⭐⭐⭐ |
| **测试定价** | 先测后定 | 新产品 | ⭐⭐⭐⭐⭐ |

### 1.3 价格锚定

**三档定价法**：

| 档位 | 价格 | 作用 |
|------|------|------|
| 入门档 | ¥99 | 降低决策门槛 |
| **标准档** | ¥299 | 主推，营收主力 |
| 高端档 | ¥999 | 锚定高端形象 |

**注意**：中间档是最重要的

### 1.4 一人公司价格参考

| 产品类型 | 价格区间 | 说明 |
|----------|----------|------|
| 提示词/模板 | ¥9.9-99 | 低价走量 |
| 小课程 | ¥99-399 | 主流 |
| 大课程 | ¥499-1999 | 需要信任 |
| 社群年费 | ¥365-999 | 持续价值 |
| 咨询/小时 | ¥300-2000 | 看专业度 |
| SaaS/月 | ¥19-199 | 订阅收入 |
| 服务/月 | ¥2000-10000 | 交付重 |

## 二、获客渠道

### 2.1 渠道分类

| 渠道 | 特点 | 一人公司适配 | 难度 |
|------|------|--------------|------|
| **小红书** | 流量大、年轻、女性为主 | ⭐⭐⭐⭐⭐ | 中 |
| **抖音/视频号** | 流量大、娱乐为主 | ⭐⭐⭐⭐ | 高 |
| **知乎** | 长图文、专业 | ⭐⭐⭐⭐ | 中 |
| **公众号** | 私域、深度 | ⭐⭐⭐⭐ | 中 |
| **B站** | 年轻、长视频 | ⭐⭐⭐ | 高 |
| **微信群** | 私域、强关系 | ⭐⭐⭐⭐ | 低 |
| **独立站/SEO** | 长期、被动 | ⭐⭐⭐ | 高 |
| **付费投放** | 快速、精准 | ⭐⭐ | 高 |
| **老带新** | 信任、成本低 | ⭐⭐⭐⭐ | 低 |

### 2.2 一人公司获客原则

**内容获客为主**：

- 成本最低（用时间换流量）
- 积累性强（内容可以长期产生流量）
- 信任度高（内容建立专业形象）

**付费投放为辅**：

- 产品验证后使用
- 有一定现金流后使用
- 计算好 ROI

### 2.3 渠道选择

**选择公式**：

```
你的目标用户 + 他们的聚集地 = 你的获客渠道
```

| 目标用户 | 推荐渠道 |
|----------|----------|
| 职场人 | 公众号、知乎 |
| 年轻女性 | 小红书 |
| 宝妈的 | 小红书、抖音 |
| 开发者 | 知乎、Twitter |
| 小老板 | 抖音、视频号 |
| 学生 | B站、小红书 |

## 三、获客执行

### 3.1 内容获客流程

```
1. 选渠道   → 确定主攻渠道
    ↓
2. 找定位   → 明确内容风格和主题
    ↓
3. 搭框架   → 建立内容体系
    ↓
4. 持续输出 → 发布内容，引流私域
    ↓
5. 转化     → 私域成交
```

### 3.2 内容体系

| 内容类型 | 作用 | 比例 |
|----------|------|------|
| **引流内容** | 吸引流量 | 40% |
| **信任内容** | 建立信任 | 40% |
| **转化内容** | 引导成交 | 20% |

### 3.3 一人公司内容策略

**聚焦 > 分散**：

- 选定 1-2 个渠道深耕
- 选定 1-2 个主题持续输出
- 统一内容风格

**质量 > 数量**：

- 一条爆款 > 十条普通
- 深度内容 > 水文
- 有价值内容 > 蹭热点

## 四、定价获客报告

### 4.1 输出模板

```markdown
## [产品名称] 定价获客方案

### 定价策略
- 定价模型：[成本/价值/竞争/心理/测试]
- 价格档位：
  - 入门档：¥xxx
  - 标准档：¥xxx（主推）
  - 高端档：¥xxx
- 定价理由：[为什么定这个价]

### 获客渠道
- 主攻渠道：[1-2 个]
- 辅助渠道：[1-2 个]
- 渠道策略：[为什么选这些渠道]

### 内容计划
- 内容主题：[3-5 个主题]
- 发布频率：[每周几条]
- 内容类型：[引流/信任/转化]

### 转化路径
- 流量来源 → [渠道]
- 私域沉淀 → [微信/社群]
- 成交转化 → [产品]
- 复购/裂变 → [策略]

### 获客成本估算
- 预计流量：[多少]
- 预计转化率：[多少]
- 预计获客成本：[多少]
```

### 4.2 案例

**案例：AI 写作课**

| 维度 | 内容 |
|------|------|
| 定价 | ¥299（标准档） |
| 主攻渠道 | 小红书 |

## 五、第一单验证

### 5.1 第一单从哪里来

| 来源 | 优先级 | 说明 |
|------|--------|------|
| 朋友圈 | ⭐⭐⭐⭐⭐ | 信任度高 |
| 私域好友 | ⭐⭐⭐⭐ | 已有信任 |
| 社群 | ⭐⭐⭐⭐ | 精准聚集 |
| 冷启动 | ⭐⭐⭐ | 付费/免费内容 |

### 5.2 第一单定价

**建议**：第一单可以稍微低价，但不要免费

| 策略 | 适用场景 |
|------|----------|
| **原价** | 验证真实付费意愿 |
| **小幅优惠** | 吸引第一批用户 |
| **限量特价** | 制造稀缺感 |

### 5.3 第一单验证清单

- [ ] 找到 10 个目标用户
- [ ] 至少 3 个表示有兴趣
- [ ] 至少有 1 个完成付费
- [ ] 验证交付能力
