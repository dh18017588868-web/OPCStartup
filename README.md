## 📌 简介 (Introduction)

OPC Startup 是一个专为一人公司创业者设计的 AI 辅助方法论系统。它整合了来自 GitHub 最佳实践的创业方法论，帮助个人创业者从 0 到 1 构建可持续的商业模式。

**核心理念**：1 名核心决策者 + N 个 AI Agent = 军团作战体系

**核心公式**：单人产值 = 决策质量 × AI放大系数 × 系统化程度

## OPC Startup - One Person Company Startup Methodology

> An integrated one-person company practice system based on best practices from the web, combining One Person Business methodology + Lean Side Business + Indie Hacker wisdom

### What is OPC?

OPC (One Person Company) is not just "working alone" in the traditional sense, but a "corps combat system" consisting of **1 core decision-maker + N AI Agents**.

**Core Formula**: Personal Output = Decision Quality × AI Amplification Factor × Systematization Level

### Why OPC Startup?

- **16 Core Skills**: From quick start to iteration optimization
- **30+ Reference Documents**: Covering all aspects of entrepreneurship
- **7-Step Startup Method**: A complete path from 0 to 1
- **AI-First**: Designed for the AI era

### Quick Start

```bash
# In CodeBuddy
输入: OPC Startup
或: /OPC-START
```

### Core Modules

| Module | Description |
|--------|-------------|
| **Mindset** | OPC definition, core philosophy |
| **Planning** | Business canvas, monthly reports |
| **Build** | MVP validation, product design |
| **Infrastructure** | Tools, automation, operations |
