# 收入追踪器模板

> 一人公司财务追踪工具，支持自动生成Markdown文件

---

## 1️⃣ 每月收入记录

```markdown
# {年份}年{月份} 收入报告

## 本月收入汇总
| 收入来源 | 金额 | 状态 |
|----------|------|------|
| 产品A | ¥X | 已到账 |
| 产品B | ¥X | 已到账 |
| 服务收入 | ¥X | 已到账 |
| 其他 | ¥X | 已到账 |
| **合计** | **¥X** | - |

## 累计收入
- 本年累计：¥X
- 首次突破：¥X (月份)

## 关键指标
- 新增付费用户：X
- 流失用户：X
- 续费率：X%
- ARPU：¥X
```

## 4️⃣ 使用方式

### 方式一：手动填写
复制上方模板到你的笔记软件（Notion/Obsidian/飞书）

### 方式二：AI 生成
告诉我：
- 你的产品名
- 本月收入数据
- 我帮你生成完整报告

### 方式三：自动化（进阶）
- 使用 Notion API 自动同步
- 使用 Stripe/Lemon Squeezy API 拉取数据
