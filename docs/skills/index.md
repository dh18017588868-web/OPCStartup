# 技能参考

OPC Startup 提供 16 个核心技能，覆盖创业全流程。

## 基础流程

| 技能 | 说明 | 触发器 |
|------|------|--------|
| [快速启动](start.md) | 引导完成七步创业流程 | `/opc-start` |
| [想法验证](validate.md) | 24小时验证市场假设 | `/opc-validate` |
| [MVP设计](mvp.md) | 设计最小可行产品 | `/opc-mvp` |
| [画布工具](canvas.md) | 填写一人企业画布 | `/opc-canvas` |
| [72小时启动](launch.md) | 快速上线计划 | `/opc-launch` |

## 决策分析

| 技能 | 说明 | 触发器 |
|------|------|--------|
| [定价指导](pricing.md) | 产品定价策略 | `/opc-pricing` |
| [决策诊断](review.md) | 极简决策检验 | `/opc-review` |
| [问题诊断](diagnosis.md) | 问题排查与解决 | `/opc-diagnosis` |

## 资源获取

| 技能 | 说明 | 触发器 |

| [工具推荐](tools.md) | 按场景推荐工具 | `/opc-tools` |

| [文档拉取](fetch.md) | 最新资讯更新 | `/opc-fetch` |

## 增强能力

| 技能 | 说明 | 触发器 |

| [代码开发](code.md) | MVP 代码实现 | `/opc-code` |
