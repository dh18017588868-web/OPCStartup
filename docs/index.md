# OPC Startup - 一人公司创业方法论

> 基于全网最佳实践整合的一人公司实战体系，融合一人企业方法论 + 精益副业 + Indie Hacker 智慧

## 🎯 核心特性

- **16个核心技能** - 从快速启动到迭代优化的完整工作流
- **30+参考文档** - 涵盖创业全过程的实用指南
- **七步创业法** - 专为一人公司设计的完整路径
- **AI优先设计** - 充分利用 AI Agent 能力

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dh18017588868-web/OPCStartup.git

# 安装依赖
pip install -r requirements.txt

# 运行验证
make validate
```

### 使用

在任何支持的 AI Agent 平台（Hermes、CodeBuddy、OpenClaw）中加载此技能包，即可使用。

常用命令：

- `/opc-start` - 快速启动引导
- `/opc-validate` - 想法验证
- `/opc-mvp` - MVP 设计
- `/opc-canvas` - 画布填写
- `/opc-launch` - 72小时快速启动
